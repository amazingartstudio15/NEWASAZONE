🔥 Step 1: Connect Firebase (Database)
Firebase stores all orders, merchants, users and riders in real-time. Free for 50,000 reads/day. Create a project at console.firebase.google.com → Web app → copy config below.
✅ Your NewasaZone Firebase project is pre-filled. Just click Connect below.
💳 Step 2: Razorpay Payments
Razorpay handles UPI, card, and wallet payments. Create account at razorpay.com → Settings → API Keys → copy Key ID.
📌 Webhook URL (copy to Razorpay dashboard)
Razorpay → Settings → Webhooks → Add webhook
https://your-domain.com/api/razorpay-webhook
💬 Step 3: WhatsApp & SMS (MSG91)
MSG91 sends WhatsApp order notifications. Sign up at msg91.com → API → copy auth key.
