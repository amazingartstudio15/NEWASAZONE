# NewasaZone

Hyperlocal super-app for Newasa, Ahilyanagar, Maharashtra (PIN 414603).

One platform for kirana, groceries, food, medicine, vegetables, and home services (plumber, electrician, carpenter, mechanic, and more) — built for Newasa taluka.

## Live app
This is a single-file HTML app (`index.html`) with no build step. Deployed via Vercel from this repo.

## Tech
- Firebase (Firestore + Auth) for live data
- Razorpay for payments
- MSG91 for WhatsApp/SMS notifications
- Leaflet (OpenStreetMap) for delivery maps

## Local testing
Just open `index.html` in any browser. No install needed.
